# InfoTrack Weather

## Prerequisites

- Get an API key from [Geoapify](https://www.geoapify.com/reverse-geocoding-api/). This will allow you to do reverse geocoding.
- Save the api key in your .env file with the key REACT_APP_REVERSE_GEOCODE_API_KEY.

## Testing

`npm run test`

## Running the app

`npm run start`
